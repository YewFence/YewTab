# Privacy Policy / 隐私政策

Last updated: 2026-04-07

This Privacy Policy explains how Yew Tab ("the Extension") handles data when you install and use the browser extension.

本隐私政策说明 Yew Tab（以下简称“本扩展”）在你安装和使用浏览器扩展时如何处理数据。

## 1. Overview / 概述

Yew Tab is a browser new-tab extension that displays and manages your browser bookmarks in a visual card layout.

Yew Tab 是一个浏览器新标签页扩展，用卡片形式展示和管理你的浏览器书签。

The Extension does not require account registration and does not operate a backend server to collect your bookmark data.

本扩展不要求注册账号，也不运营用于收集你的书签数据的后端服务器。

## 2. Data We Access / 我们会访问哪些数据

To provide its core features, the Extension may access the following data stored in your browser:

为了提供核心功能，本扩展可能会访问以下存储在浏览器中的数据：

- Bookmark data, including bookmark titles, URLs, folder names, and folder structure
- Extension settings, such as layout preferences, pinned items, startup folder, search engine preference, theme settings, and background settings
- User-selected local image files when you choose to upload a custom background image
- User-provided image URLs when you choose to use a remote image as a custom background
- Search keywords you enter when using the extension's web search feature

- 书签数据，包括书签标题、URL、文件夹名称和文件夹结构
- 扩展设置，例如布局偏好、固定项目、启动文件夹、搜索引擎偏好、主题设置和背景设置
- 当你主动上传自定义背景图时，你选择的本地图片文件
- 当你主动使用网络图片作为背景时，你提供的图片 URL
- 当你主动使用扩展内网页搜索功能时，你输入的搜索关键词

## 3. How Data Is Used / 数据如何被使用

The Extension uses the accessed data only to provide user-facing functionality, including:

本扩展仅将访问到的数据用于实现面向用户的功能，包括：

- Displaying and organizing bookmarks on the new tab page
- Creating, editing, moving, copying, deleting, importing, and exporting bookmarks when requested by you
- Saving local preferences and UI state
- Applying custom backgrounds and theme settings
- Opening web searches using the search engine selected by you

- 在新标签页展示和整理书签
- 在你主动操作时创建、编辑、移动、复制、删除、导入和导出书签
- 保存本地偏好设置和界面状态
- 应用自定义背景和主题设置
- 使用你选择的搜索引擎发起网页搜索

## 4. Local Storage / 本地存储

The Extension stores bookmark snapshots and user preferences locally in the browser's extension storage area on your device.

本扩展会将书签快照和用户偏好保存在你设备上的浏览器扩展本地存储区域中。

This local storage may include:

本地存储内容可能包括：

- Cached bookmark tree data
- Layout and navigation state
- Search engine preference
- Theme and background settings
- Uploaded custom background image data

- 缓存的书签树数据
- 布局与导航状态
- 搜索引擎偏好
- 主题和背景设置
- 你上传的自定义背景图片数据

## 5. Data Transmission / 数据传输

The Extension does not send your bookmark data to the developer's own servers.

本扩展不会将你的书签数据发送到开发者自有服务器。

However, certain features may involve requests to third-party services as part of normal operation or at your direction:

但在正常运行或你主动触发某些功能时，扩展可能会向第三方服务发起请求：

- When you use the web search feature, your search query is sent to the search engine you selected, such as Google, Bing, or DuckDuckGo.
- When you use a remote image URL as a custom background, the browser requests that image from the URL you provided.
- Website icons may be fetched through browser-provided favicon mechanisms, and in limited compatibility scenarios may rely on third-party favicon services using domain information.

- 当你使用网页搜索功能时，你的搜索词会被发送到你所选择的搜索引擎，例如 Google、Bing 或 DuckDuckGo。
- 当你使用网络图片 URL 作为自定义背景时，浏览器会向你提供的图片地址请求该图片。
- 网站图标可能通过浏览器提供的 favicon 机制获取；在少数兼容性场景下，也可能基于域名信息请求第三方 favicon 服务。

## 6. Data Sharing / 数据共享

The developer does not sell, rent, or broker your personal data.

开发者不会出售、出租或交易你的个人数据。

Data is not shared with third parties except as necessary to perform features you explicitly use, as described above.

除上文所述、为实现你主动使用的功能所必需的情形外，开发者不会与第三方共享你的数据。

## 7. Data Retention / 数据保留

Data stored by the Extension remains in your browser until:

由扩展存储在本地的数据会一直保留在你的浏览器中，直到出现以下情况之一：

- You change or delete it
- You reset the Extension's settings
- You remove the Extension
- Your browser clears the extension's local storage

- 你主动修改或删除它
- 你重置扩展设置
- 你卸载本扩展
- 浏览器清除了扩展本地存储

## 8. Your Choices / 你的选择

You may control data handling by:

你可以通过以下方式控制数据处理：

- Choosing whether to install or remove the Extension
- Choosing whether to upload a local background image
- Choosing whether to use an external image URL
- Choosing whether to use the web search feature
- Editing, exporting, or deleting bookmarks and extension settings

- 选择是否安装或卸载本扩展
- 选择是否上传本地背景图片
- 选择是否使用外部图片 URL
- 选择是否使用网页搜索功能
- 编辑、导出或删除书签及扩展设置

## 9. Children / 儿童

The Extension is not directed to children under the age required by applicable law, and the developer does not knowingly collect personal information from children.

本扩展不面向适用法律规定年龄以下的儿童，开发者也不会故意收集儿童个人信息。

This Privacy Policy may be updated from time to time to reflect product changes, legal requirements, or operational updates. The latest version will be published at the same URL.

本隐私政策可能会根据产品变化、法律要求或运营更新而调整。最新版本会发布在同一链接下。
