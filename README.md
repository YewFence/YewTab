# yew-tab

`yew-tab` 是一个基于 React 和 Vite 构建的现代浏览器扩展（Manifest V3），旨在为 Edge 和 Chrome 提供一个干净、高效的新标签页体验。

它主要专注于优化的书签管理和流畅的用户界面。

## ✨ 特性

- **自定义新标签页**: 替换默认的新标签页，提供更清爽的界面。
- **书签管理**: 直接在新标签页浏览和管理你的书签。
- **现代化技术栈**: 使用 React 18, TypeScript 和 Vite 构建。
- **高性能**: 快速启动，轻量级。
